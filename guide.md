# SiBantu

Sistem Informasi Pengaduan Masyarakat Untuk Penyaluran Bantuan Bencana Berbasis Web.

## Teknologi

- Laravel 12
- MySQL
- Tailwind CSS
- Alpine.js
- Leaflet.js
- Chart.js
- DomPDF

## Instalasi

```bash
git clone https://github.com/RafiqMuzayyan/sibantu.git

cd sibantu

composer install

cp .env.example .env

php artisan key:generate

php artisan migrate --seed

npm install

npm run build

php artisan serve
```
